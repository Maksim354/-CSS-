Ласкаво просимо на першу сторінку!

Ця сторінка використовує зовнішні стилі, які підключені через окремий файл styles.css.

Список цікавих фактів:

  • Зовнішні стилі легко підтримувати.
  • Можна використовувати один файл для декількох сторінок.
  • Оновлення стилів впливає на всі сторінки одночасно.
